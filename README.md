# 🩺 PulseScore: Executive Healthcare Decision Intelligence

An enterprise-grade Executive Healthcare Dashboard built with Streamlit, featuring dynamic Surge Risk Intelligence (SRI) modeling, national infrastructure snapshots, and gap diagnosis for the Indian healthcare sector.

---

## 🚀 Core Features

- **National Snapshot** — Real-time overview of healthcare KPIs (Beds, Personnel, ICU Reach) across States and UTs.
- **Surge Risk Intelligence (SRI)** — Predicts system collapse points using weighted demand-supply variables.
- **Structural Gap Diagnosis** — Identifies resource deficits using global benchmarks.
- **Nearest Hospital Finder** — Geospatial search and district-level exploration.
- **Enterprise Reporting** — One-click PDF intelligence report generation.
