## 📌 Overview
**PulseScore** is an enterprise-grade healthcare analytics dashboard that transforms raw healthcare infrastructure data into **decision intelligence insights** for policymakers, administrators, and analysts.

It converts complex datasets into interactive dashboards, enabling faster and smarter decisions about healthcare resource allocation, risk detection, and infrastructure planning.

---

## 🎯 Key Objectives
- Provide centralized healthcare analytics platform
- Identify infrastructure gaps
- Detect surge risk levels
- Support policy planning decisions
- Visualize regional healthcare distribution
